# Literature Review

## Installation

### Install Poetry

If poetry is not installed,

```bash
pip install pipx
pipx install poetry
```

### Install Repo with Poetry

```bash
git clone https://github.com/clinical-trials-research/literature-review.git
cd literature-review
poetry install
```
